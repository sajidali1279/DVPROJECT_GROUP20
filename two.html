Stringency Index vs Number of Accidents

This graph is multi line graph where each color represents a number or trend. Blue color represents the number of causalities and purple color represents the stringency index. Using this we can observe whether there is any correlation between stringency index and the number of causalities. We do not observe any particular pattern but at some places we can see that when the strigency index is dropped the number of causalities dropped.
