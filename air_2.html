Number of Fatalities due to Airplane Crashes in each Country

This visualization is a map which represents the number of fatalities due to airplane crashes for each country over the period. For each location coloured in the graph, we have the number of fatalities and this can be obtained by hovering over the graph. For example, Russia is represented in pale teal color and the total number of fatalities are 558. We can get the fatalities for each country and can be useful for further analysis.
